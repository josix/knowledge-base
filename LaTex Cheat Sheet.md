# LaTex Cheat Sheet

## Math Symbol

- 草寫符號
  - $$\mathcal{A}$$

- 實數空間
  - $$\mathbb{R}$$

- 集合

  - 子集

    - $$\subset$$

  - 映射

    - $$\rightarrow$$

- 運算符號
  - 求和
    - $$\sum_{i}^{n}$$
  - Argmin, Argmax
    - $$\mathop{\arg\max}_\theta$$
  - 分數
    - $$\frac{x}{y}$$
  - Norm
    - $$\|x\|$$

## Syntax

- 條列項目

  - 無序

    ```latex
    \begin{itemize}
      \item One
      \item Two
      \item Three
    \end{itemize}
    ```
  
  - 有序
  
    ```latex
    \begin{enumerate}
      \item One
      \item Two
      \item Three
    \end{enumerate}
    ```
